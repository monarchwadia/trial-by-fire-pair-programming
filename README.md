# Requirements

- We will build a RESTful API
- It will have 5 routes
- GET list all the items
- POST to create an item
- PUT/PATCH to edit an item
- GET to get a specific item
- DELETE to delete an item

What is an item? A Todo List item.

- id
- createdAt
- title
- subject
- body
